Changements effectués le 2026-05-14
============================================================

Contexte : permettre la cohabitation de deux apps sur le robot
Temi — celle de Johan (package com.temi.temi_robot, signature
1c77046c, installée le 2025-07-22) et la version de test pour

Approche retenue : utiliser un applicationIdSuffix sur le
build type debug pour que le package devienne
com.temi.temi_robot.test → Android voit deux apps distinctes.

------------------------------------------------------------
Limites connues de la build "Temi Test"
------------------------------------------------------------

Le SDK Temi (carte, patrol, goTo, meetings) est enregistré
dans la console développeur Temi avec le package name
com.temi.temi_robot + l'empreinte SHA d'origine. La build
.test ne sera donc PAS reconnue par le robot pour ces
fonctionnalités.

Fonctionne sur la build .test :
- Wave gesture (CameraX + MediaPipe + TTS "Hello!")
- UI, pages, sauvegarde JSON locale

Ne fonctionne PAS sur la build .test :
- Navigation (goTo, patrol, goToHomeBase)
- Meetings (appel librarian Kamil)
- Permissions Temi (MAP, MEETINGS)

    @SuppressLint("UnsafeOptInUsageError")
    private fun bindCamera() {
        val providerFuture = ProcessCameraProvider.getInstance(context)
        providerFuture.addListener({
            try {
                cameraProvider = providerFuture.get()

                // Diagnostic: list every camera CameraX can see on this device.
                val cams = cameraProvider?.availableCameraInfos ?: emptyList()
                Log.i(TAG, "Available cameras: ${cams.size}")
                cams.forEachIndexed { i, info ->
                    val lensName = when (info.lensFacing) {
                        CameraSelector.LENS_FACING_FRONT -> "FRONT"
                        CameraSelector.LENS_FACING_BACK -> "BACK"
                        CameraSelector.LENS_FACING_EXTERNAL -> "EXTERNAL"
                        else -> "UNKNOWN(${info.lensFacing})"
                    }
                    Log.i(TAG, "  cam[$i] lensFacing=$lensName")
                }

                // Prefer FRONT (selfie cam); fall back to BACK if FRONT not exposed.
                val selector: CameraSelector = when {
                    cameraProvider?.hasCamera(CameraSelector.DEFAULT_FRONT_CAMERA) == true -> {
                        Log.i(TAG, "Using FRONT camera")
                        CameraSelector.DEFAULT_FRONT_CAMERA
                    }
                    cameraProvider?.hasCamera(CameraSelector.DEFAULT_BACK_CAMERA) == true -> {
                        Log.w(TAG, "FRONT camera not available, falling back to BACK camera")
                        CameraSelector.DEFAULT_BACK_CAMERA
                    }
                    else -> {
                        Log.e(TAG, "No camera available at all — aborting bind")
                        return@addListener
                    }
                }

                val analysis = ImageAnalysis.Builder()
                    .setBackpressureStrategy(ImageAnalysis.STRATEGY_KEEP_ONLY_LATEST)
                    .build()
                    .also {
                        it.setAnalyzer(cameraExecutor) { imageProxy ->
                            processImageProxy(imageProxy)
                        }
                    }

                cameraProvider?.unbindAll()
                cameraProvider?.bindToLifecycle(
                    lifecycleOwner,
                    selector,
                    analysis,
                )
                Log.i(TAG, "Camera bound + analysis use case active")
            } catch (e: Exception) {
                Log.e(TAG, "Failed to bind camera", e)
            }
        }, ContextCompat.getMainExecutor(context))
    }

    private fun processImageProxy(imageProxy: ImageProxy) {
        try {
            framesProcessed++
            if (framesProcessed == 1L || framesProcessed % 60L == 0L) {
                Log.i(TAG, "Camera frames processed: $framesProcessed (hand-detected: $handsDetected)")
            }
            val bitmap: Bitmap = imageProxy.toBitmap()
            val mpImage = BitmapImageBuilder(bitmap).build()
            handLandmarker?.detectAsync(mpImage, System.currentTimeMillis())
        } catch (e: Exception) {
            Log.e(TAG, "Frame processing failed", e)
        } finally {
            imageProxy.close()
        }
    }

    private fun handleResult(result: HandLandmarkerResult) {
        if (result.landmarks().isEmpty()) return
        handsDetected++
        if (!firstHandLogged) {
            firstHandLogged = true
            Log.i(TAG, "First hand detected")
        }
        // Landmark 0 = wrist; .x() is normalized to [0,1] across frame width
        val wristX = result.landmarks()[0][0].x()

- On verra dans logcat la liste exacte des caméras exposées

- Si la front cam n'est pas dispo, on bascule automatiquement

  qui pointe vers l'utilisateur depuis le robot).
- On verra défiler les frames traitées et la détection de
  main pour distinguer "caméra pas bindée" / "caméra OK mais
  pas de main détectée" / "main détectée mais wave non
